# NLP_s2s

## 作業概要
此次作業為利用 已有的 word embeddings，使用 NLTK 與 jieba 進行中英文分詞，再以 LSTM 的設計，訓練出英翻中的 sequence-to-sequence 深度學習模型。

## 測試結果

了 不会 一個 不 小 。
